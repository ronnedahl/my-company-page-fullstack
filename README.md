# WEBBkraft — webbkraft.net

Full-stack web agency site with booking system and AI chatbot.
Built as a solo project to demonstrate real-world capabilities.

## Features
- Multi-step booking system with SQL database
- AI chatbot trained on site content (RAG)
- SEO-optimized pages
- Mobile-responsive design

## Tech Stack
| Layer | Technology |
|-------|-----------|
| Frontend | Next.js 14, TypeScript, Tailwind CSS |
| Backend | Python, FastAPI |
| Database | MySQL |
| AI | OpenAI, LangChain |

## Live Demo
[webbkraft.net](https://webbkraft.net) — Booking demo at [webbkraft.net/booking](https://webbkraft.net/booking)

## Note
This is a production project, not an open-source template.
Local setup requires private environment variables and database access.
