här är riktlinjer som man kan följa för hur stor en mobils text ska vara.

body {
  font-size: 1rem;      /* 16px */
}

h1 {
  font-size: 2rem;      /* 32px */
}

h2 {
  font-size: 1.5rem;    /* 24px */
}

h3 {
  font-size: 1.25rem;   /* 20px */
}

h4 {
  font-size: 1.125rem;  /* 18px */
}

small {
  font-size: 0.875rem;  /* 14px */
}
